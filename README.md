# ToDo List by TucanoWeb with NextJS

Project made with one of my students, using NextJS



# Get Starter



## Install Dependencies

In the root of the project, run the command:

```bash
npm install
```



## Run Project

Yet in the root of the project, run the command:

```bash
npm run dev
```
